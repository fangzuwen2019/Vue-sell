## 安装使用

``` bash
# 从仓库中下载
$ git clone https://github.com/icharle/Vue-sell.git


# 进入代码根目录安装依赖
$ npm install


# 本地运行调试
$ npm run dev


# 打包压缩发布项目
$ npm run build


# 打包发布后根目录会生成dist文件夹，该文件夹即为项目的所有代码(webpack自动为我们压缩打包)
```
